# دليل تشغيل النظام على السيرفر الخاص (beeoons.com)

لقد قمت بتحديث جميع الأكواد لتعمل مع الدومين الجديد `https://beeoons.com`. إليك الخطوات المطلوبة لتشغيل النظام:

## 1. تجهيز قاعدة البيانات (Database)
1. قم بإنشاء قاعدة بيانات MySQL جديدة على السيرفر.
2. قم بتشغيل الأوامر الموجودة في ملف `backend/DATABASE_SYNC.sql` لإنشاء الأعمدة المطلوبة.
3. تأكد من تحديث بيانات الاتصال بقاعدة البيانات في ملف `.env` داخل مجلد `backend`.

## 2. تشغيل الباكيند (Backend)
1. ادخل إلى مجلد `backend` عبر الـ Terminal في السيرفر.
2. قم بتثبيت المكتبات: `npm install`
3. قم ببناء المشروع: `npm run build`
4. قم بتشغيل المشروع باستخدام PM2 لضمان استمراريته:
   ```bash
   pm2 start dist/main.js --name bnpl-backend
   ```

## 3. تشغيل لوحة التحكم (Admin & Vendor)
1. بعد تحديث الروابط في الكود (الذي قمت به بالفعل)، قم بعمل `npm run build` لكل من `bnpl-admin` و `bnpl-vender`.
2. ارفع محتويات مجلد `out` أو `dist` الناتج إلى المسار المطلوب في `public_html`.

## 4. تطبيق الموبايل (Flutter)
1. لقد قمت بتحديث الرابط في ملف `lib/config/env/env_dev.dart`.
2. يمكنك الآن إعادة بناء التطبيق (Build APK/IPA) وسيعمل مع السيرفر الجديد مباشرة.

> [!NOTE]
> تأكد من أن السيرفر يسمح بمرور (CORS) من الدومينات الخاصة بلوحة التحكم.
