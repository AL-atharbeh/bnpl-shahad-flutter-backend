# BNPL - اشتر الآن وادفع لاحقاً

تطبيق Flutter لتجربة BNPL (Buy Now Pay Later) مع دعم كامل للغة العربية والإنجليزية.

## المميزات

- 🌍 دعم كامل للترجمة (العربية والإنجليزية)
- 🎨 تصميم حديث باستخدام Material Design 3
- 📱 واجهة مستخدم متجاوبة
- 🚀 صفحات ترحيب تفاعلية
- 🔧 بنية نظيفة ومنظمة (Clean Architecture)

## متطلبات التشغيل

- Flutter SDK 3.6.0 أو أحدث
- Dart SDK 3.6.0 أو أحدث

## كيفية التشغيل

1. **تثبيت التبعيات:**
   ```bash
   flutter pub get
   ```

2. **توليد ملفات الترجمة:**
   ```bash
   flutter gen-l10n
   ```

3. **تشغيل التطبيق:**
   ```bash
   flutter run
   ```

## بنية المشروع

```
lib/
├── config/                 # إعدادات التطبيق
│   ├── env/               # ملفات البيئة
│   └── constants/         # الثوابت
├── features/              # ميزات التطبيق
│   └── onboarding/        # ميزة الترحيب
│       ├── presentation/  # طبقة العرض
│       ├── data/          # طبقة البيانات
│       └── domain/        # طبقة المنطق
├── l10n/                  # ملفات الترجمة
│   ├── arb/              # ملفات ARB
│   └── generated/        # ملفات الترجمة المولدة
└── main.dart             # نقطة البداية
```

## المسارات المتاحة

- `/onboarding` - صفحة الترحيب
- `/auth` - صفحة تسجيل الدخول (قيد التطوير)
- `/home` - الصفحة الرئيسية (قيد التطوير)

## الترجمة

التطبيق يدعم اللغتين العربية والإنجليزية مع إعداد العربية كلغة افتراضية.

### إضافة ترجمة جديدة:

1. أضف المفتاح في `lib/l10n/arb/app_en.arb`
2. أضف الترجمة العربية في `lib/l10n/arb/app_ar.arb`
3. شغل `flutter gen-l10n` لتوليد الملفات
