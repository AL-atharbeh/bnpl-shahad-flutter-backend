# ميزة تأجيل القسط المجاني

## نظرة عامة
تم إضافة ميزة جديدة تسمح للمستخدمين بتأجيل قسط واحد مجاناً في حال واجهوا ظرفاً طارئاً. هذه الميزة متاحة **مرة واحدة فقط** لكل قسط.

## المميزات الرئيسية

### 1. **Badge التأجيل المجاني**
- يظهر على بطاقات الأقساط التي يمكن تأجيلها مجاناً
- تصميم بلون ذهبي مميز مع أيقونة هدية
- النص: "تأجيل مجاني • مرة واحدة"

### 2. **زر التأجيل المجاني**
- يظهر كزر بارز في أعلى أزرار البطاقة
- تصميم خاص بألوان ذهبية متدرجة
- يحتوي على أيقونة هدية بجانب النص

### 3. **نافذة التأكيد (Bottom Sheet)**
تحتوي على:
- **أيقونة وعنوان**: أيقونة التقويم مع عنوان "تأجيل قسط واحد مجاناً"
- **Badge**: "تأجيل مجاني • مرة واحدة"
- **الوصف**: شرح الميزة بأنها بدون رسوم إضافية
- **تفاصيل القسط**: 
  - اسم المتجر والمبلغ
  - تاريخ الاستحقاق الحالي (مشطوب)
  - تاريخ الاستحقاق الجديد (بعد 30 يوماً)
- **ملاحظة**: تنبيه بأن الميزة تُستخدم مرة واحدة فقط
- **أزرار**: إلغاء وتأكيد التأجيل

### 4. **تتبع الاستخدام**
- يتم حفظ الأقساط المؤجلة محلياً باستخدام `SharedPreferences`
- كل قسط له معرّف فريد: `merchantName_installmentNumber`
- بمجرد استخدام التأجيل المجاني لقسط معين، لا يمكن استخدامه مرة أخرى

## البنية التقنية

### الملفات الجديدة

#### 1. `lib/services/postpone_service.dart`
خدمة إدارة ميزة التأجيل المجاني:

**الخصائص:**
- `isInitialized`: حالة تهيئة الخدمة
- `postponedInstallments`: قائمة الأقساط المؤجلة
- `postponedCount`: عدد الأقساط المؤجلة

**الدوال:**
- `initialize()`: تهيئة الخدمة وتحميل البيانات المحفوظة
- `canPostponeForFree(installmentId)`: التحقق من إمكانية التأجيل المجاني
- `postponeForFree(...)`: تنفيذ التأجيل وحفظ البيانات
- `cancelPostpone(installmentId)`: إلغاء التأجيل (للاختبار)
- `clearAll()`: مسح كل البيانات (للاختبار)

#### 2. `lib/features/payments/presentation/widgets/free_postpone_sheet.dart`
نافذة تأكيد التأجيل المجاني:

**المكونات:**
- عرض معلومات القسط والتواريخ
- رسائل توضيحية وتحذيرات
- أزرار تأكيد وإلغاء
- تصميم احترافي متناسق مع التطبيق

## التصميم البصري

### الألوان المستخدمة
- **الذهبي**: `#F59E0B` للأيقونات والنصوص
- **الخلفية الذهبية**: `#FFF4E6` و `#FFFBEB` (gradient)
- **الحدود**: `#FEF3C7`
- **الأخضر**: `AppColors.primary` للزر الأساسي
- **الأزرق المعلوماتي**: `#0284C7` للرسائل التوضيحية

### التصميم
- **Badge**: مستطيل مستدير مع gradient ذهبي
- **زر التأجيل**: زر كامل العرض مع gradient وأيقونة
- **Bottom Sheet**: تصميم نظيف مع sections واضحة
- **الأيقونات**: استخدام Material Icons المناسبة

## تدفق العمل (Flow)

```
1. المستخدم يرى بطاقة القسط
   ↓
2. إذا كان التأجيل المجاني متاح:
   - يظهر Badge ذهبي
   - يظهر زر "تأجيل مجاناً" في الأعلى
   ↓
3. المستخدم ينقر على "تأجيل مجاناً"
   ↓
4. تظهر نافذة التأكيد مع:
   - تفاصيل القسط
   - التواريخ (الحالي والجديد)
   - ملاحظة الاستخدام مرة واحدة
   ↓
5. المستخدم يؤكد التأجيل
   ↓
6. يتم:
   - حفظ القسط في قائمة المؤجلة
   - إخفاء Badge والزر من البطاقة
   - عرض رسالة نجاح
```

## الاستخدام في الكود

### التحقق من إمكانية التأجيل
```dart
final postponeService = Provider.of<PostponeService>(context);
final installmentId = '${merchantName}_${installmentNumber}';
final canPostpone = postponeService.canPostponeForFree(installmentId);
```

### تنفيذ التأجيل
```dart
await postponeService.postponeForFree(
  installmentId,
  merchantName: 'Boutiqaat',
  amount: 4.060,
  originalDueDate: '15 نوفمبر 2024',
  newDueDate: '15 ديسمبر 2024',
);
```

### عرض نافذة التأكيد
```dart
FreePostponeSheet.show(
  context,
  merchantName: 'Boutiqaat',
  amount: 4.060,
  currentDueDate: '15 نوفمبر 2024',
  newDueDate: '15 ديسمبر 2024',
  onConfirm: () {
    // التأجيل تم تأكيده
  },
);
```

## التخزين المحلي
- **المفتاح**: `postponed_installments`
- **النوع**: `List<String>` (معرّفات الأقساط المؤجلة)
- **المكتبة**: `shared_preferences`

## الاختبار

### للاختبار يدوياً:
1. شغّل التطبيق
2. اذهب إلى صفحة الأقساط المعلقة
3. ابحث عن بطاقة قسط عليها Badge "تأجيل مجاني • مرة واحدة"
4. اضغط على زر "تأجيل مجاناً"
5. تحقق من:
   - ظهور النافذة بشكل صحيح
   - عرض التواريخ بشكل صحيح
   - التأكيد يعمل
   - اختفاء Badge بعد التأجيل

### لمسح البيانات (للاختبار):
```dart
final postponeService = Provider.of<PostponeService>(context, listen: false);
await postponeService.clearAll();
```
