# نظام نقاط المكافآت (Rewards Points System)

## 📋 نظرة عامة

تم إضافة نظام نقاط المكافآت الذي يربط عمليات الدفع بالنقاط المكتسبة. كل عملية دفع تعطي المستخدم نقاطاً يمكن استبدالها لاحقاً بخصومات ومكافآت.

---

## ✨ المميزات الرئيسية

### 1. **كسب النقاط التلقائي**
- ✅ كل 1 دينار مدفوع = 1 نقطة مكتسبة
- ✅ النقاط تُضاف تلقائياً بعد كل عملية دفع ناجحة
- ✅ إشعار فوري للمستخدم يعرض النقاط المكتسبة

### 2. **عرض النقاط في الملف الشخصي**
- 📊 بطاقة جميلة بتصميم عصري (Gradient Purple-Indigo)
- 📈 عرض النقاط الحالية بشكل واضح وكبير
- 💰 عرض القيمة المالية للنقاط (100 نقطة = 1 دينار)
- 📜 سجل كامل لكل عمليات النقاط

### 3. **سجل النقاط (Points History)**
- 🕐 عرض جميع عمليات كسب واستبدال النقاط
- 📅 تنسيق التاريخ بشكل ذكي (اليوم، أمس، منذ X أيام)
- 💵 ربط كل عملية بمبلغ الدفع المقابل
- 🎨 تصميم جذاب مع أيقونات واضحة

### 4. **استبدال النقاط (قيد التطوير)**
- 🎁 زر لاستبدال النقاط (يظهر عند امتلاك 100 نقطة أو أكثر)
- 🔄 الميزة قيد التطوير للربط بالعروض والخصومات

## 🏗️ البنية التقنية

### الملفات الرئيسية

#### 1. خدمة النقاط (`points_service.dart`)
```dart
lib/services/points_service.dart
```

**الوظائف الأساسية:**
- `initialize()` - تهيئة الخدمة وتحميل البيانات
- `addPointsFromPayment()` - إضافة نقاط من عملية دفع
- `calculatePointsFromPayment()` - حساب النقاط من مبلغ الدفع
- `redeemPoints()` - استبدال النقاط
- `getPointsValue()` - حساب القيمة المالية للنقاط

**التخزين:**
- يستخدم `SharedPreferences` لتخزين النقاط محلياً
- يحفظ النقاط الحالية، الإجمالي، وسجل العمليات

#### 2. واجهة النقاط في الملف الشخصي
```dart
lib/features/profile/presentation/pages/profile_page.dart
```

**المكونات:**
- `RewardsPointsCard` - البطاقة الرئيسية لعرض النقاط
- `_PointsTransactionItem` - عنصر عملية نقاط في السجل
- `_showPointsHistory()` - عرض سجل النقاط في Bottom Sheet

#### 3. الربط بعمليات الدفع
```dart
lib/features/payments/presentation/pages/payments_page.dart
```

**التحديث:**
- `_handleSuccessfulPayment()` - معالجة الدفع الناجح وإضافة النقاط
- عرض إشعار بالنقاط المكتسبة مع رسالة النجاح

#### 4. ملفات الترجمة
```
lib/l10n/arb/app_ar.arb
lib/l10n/arb/app_en.arb
```

**النصوص المضافة:**
- `rewardsPoints` - نقاط المكافآت
- `yourPoints` - نقاطك
- `earnPointsWithPayments` - اكسب نقاط مع كل عملية دفع
- `pointsHistory` - سجل النقاط
- `earnedPoints` - نقاط مكتسبة
- `redeemedPoints` - نقاط مستبدلة
- `earnedPointsMessage` - رسالة النقاط المكتسبة
- وغيرها...

## 📊 معدلات الكسب والاستبدال

### الكسب
```
1 دينار مدفوع = 1 نقطة
مثال: دفع 32.185 دينار = 32 نقطة
```

### الاستبدال (قيمة المكافأة)
```
100 نقطة = 1 دينار
مثال: 500 نقطة = 5 دينار خصم
```

## 🔄 تدفق العمل (Workflow)

```
1. المستخدم يقوم بعملية دفع
   ↓
2. تنفيذ الدفع بنجاح
   ↓
3. PointsService.addPointsFromPayment()
   ↓
4. حساب النقاط (amount × 1.0)
   ↓
5. إضافة النقاط إلى رصيد المستخدم
   ↓
6. حفظ العملية في السجل
   ↓
7. عرض إشعار بالنقاط المكتسبة
   ↓
8. تحديث الواجهة في الملف الشخصي
```
